اضغط على الزر أدناه لإضافة مجموعة من الطلبات التجريبية لاختبار واجهة الأدمن. يمكنك مسحها في أي وقت من صفحة الطلبات.

💡 تكامل الموقع الرئيسي: الطلبات الحقيقية تُرسل تلقائياً عند ضغط "إرسال الطلب" في نموذج التواصل.

كود التكامل مع الصفحة الرئيسية:

function submitForm() {
  // ... validation ...
  const order = {
    id: Date.now(),
    name: fName.value,
    phone: fPhone.value,
    city: fCity.value,
    plan: fPlan.value,
    payment: fPayment.value,
    message: fMsg.value,
    date: new Date().toISOString(),
    status: 'new'
  };
  const orders = JSON.parse(
    localStorage.getItem('netpros_orders') || '[]'
  );
  orders.unshift(order);
  localStorage.setItem(
    'netpros_orders', JSON.stringify(orders)
  );
}
